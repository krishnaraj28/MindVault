## MicroProject

### Overview
**MindVault** is a Python-based graphical flashcard application designed for learning and reviewing concepts effectively. The application leverages **Tkinter** for the GUI, **JSON** for data storage, and **Pillow** for handling images. Users can add, edit, delete, and review flashcards, making it a flexible and user-friendly learning tool.

###üöÄ Features

- **Home Screen**:  
  A visually appealing home screen with a background image and application title.

- **Flashcard Management**:
  - Add new flashcards with custom topics and definitions.
  - Edit existing flashcards.
  - Delete flashcards.
  - Save and load flashcards persistently using a JSON file.

- **Flashcard Review**:
  - View all flashcards in a grid format with randomized colors.
  - Navigate through flashcards using "Next" and "Previous" buttons.
  - Flip flashcards to reveal definitions or topics.

- **Custom Design**:
  - Randomized background colors for each flashcard.
  - Enhanced user experience with consistent button and label styles.

- **Responsive Layout**:  
  The app dynamically adjusts to various screen sizes and supports zoomed window states.
